Docker Note

What is Docker

  1. Docker is a tool for running applications in an isolated environment

Container vs VM

  1. Containers are an abstraction at the app layer that packerages code and dependencies together. Multiple containers can run on the same machine and share the OS kernel with other containers, each running as isolated processes in user space.
  2. Virtual machines are an abstraction of physical hardware turning one server into many servers. The hypervisor allows multiple VMs to run on a single machine. Each VM includes a full copy of an operating system, the application, necessary binary and libraries - taking up tens of GBs.
